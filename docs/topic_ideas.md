# Topic Ideas

## 候选方向 1：研发资本化率抽取与横向比较

- **公告类型**：年度报告
- **关键词**：研发支出、资本化、费用化
- **金融问题**：不同行业、不同公司的研发资本化率有何差异？是否存在异常高的公司？
- **可能字段**：研发支出总额、资本化金额、费用化金额、资本化率、行业分类
- **预计输出**：各公司资本化率排名、行业对比分析
- **主要风险**：
  - 数据量相对单一（仅年报）
  - 缺少外部验证（无法判断"异常"是否真实存在）
  - 任务复杂度可能不足以达到挑战档满分

## 候选方向 2：研发资本化风险排序与问询函触发机制分析

- **公告类型**：年度报告 + 问询函 + 回复函
- **关键词**：研发、资本化、开发支出、问询函
- **金融问题**：什么样的资本化特征最容易触发监管问询？能否基于年报数据提前识别异常？
- **可能字段**：14+ 核心财务字段 + 8+ 问询函关联字段
- **预计输出**：异常公司列表、四维度评分、混淆矩阵、闭环评估报告
- **主要风险**：
  - 问询函数量可能不足
  - 增加问询函查询和解析的工作量
  - 时间线关联逻辑较复杂

## 初步排序

1. **候选方向 2**（首选）
   - 理由：任务复杂度更高，覆盖挑战档全部维度；问询函闭环提供外部监管信号，可做可行性测试（受正样本量限制，非严格验证）；与参考选题池#10有差异化
2. **候选方向 1**（备选）
   - 理由：如果问询函数据不足可退回到此方案；基础数据准备相同，不影响前期进度
